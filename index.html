Malscan is an intuitive and easy to use malware scanner for linux, extending ClamAV with additional scanning modes and malware signatures.

Any Linux, Any Time

Malscan is pre-packged for use with Red Hat Enterprise Linux and CentOS operation systems, however it can quickly be installed from source on nearly any Linux operating system that exists. We regularly test Malscan on a number of the most widely used operating systems, including Ubuntu, Debian, Arch Linux, Fedora, and OpenSUSE to ensure that whatever your use case you will be protected.

Open And Free

Malscan is free and open source software, released under the MIT License. You can inspect every line of code in the program and make any changes that you'd like, guaranteeing that you'll always be safe and Malscan will always be reliable and transparent.
