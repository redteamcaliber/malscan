Malscan Documentation

Malscan's documentation is an ongoing process. You can contribute to the documentation by creating a pull request to the gh-pages branch of the Malscan repository. If you come across any issues with this documentation, or if you find anything that could be made more clear, please submit an issue in GitHub using the documentation tag.
